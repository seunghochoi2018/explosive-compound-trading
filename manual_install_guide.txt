Ollama 수동 설치 가이드
========================

1. 웹사이트 방문
   - https://ollama.com/download/windows
   - "Download for Windows" 클릭

2. 설치 파일 실행
   - 다운로드된 ollama-windows-amd64.exe 실행
   - 관리자 권한으로 실행

3. PATH 설정 확인
   - 설치 경로: C:\Users\user\AppData\Local\Programs\Ollama
   - 이 경로가 PATH에 자동 추가되는지 확인

4. 설치 확인
   - 새 PowerShell 창 열기
   - ollama --version 명령 실행
   - 버전 정보가 나오면 성공

5. 모델 다운로드
   - ollama pull qwen2.5:7b
   - ollama pull qwen2.5:14b

6. 서비스 시작
   - ollama serve
   - 포트 11434에서 실행 확인
